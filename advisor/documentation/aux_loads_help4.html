Auxiliary Loads Help

Manage List

Overview

The “Manage List” figure is used to create, edit, and delete auxiliary load files.  Creating a new load file brings up another figure to define a new load.  The default data in the new figure is based upon the load file highlighted in the “Manage List” figure.  Editing and deleting load files can only be done with files generated using the “Create” button.  This preserves the original ADVISOR load file data.
