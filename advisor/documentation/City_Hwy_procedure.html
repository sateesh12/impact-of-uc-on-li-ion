City Hwy Test Procedure

Test Procedure Description
The City Highway Test Procedure runs a cold start FTP-75 cycle and a hot-start HWFET cycle.  If TEST_CITY_HWY_HYBRID is selected then 2 complete UDDS cycles are run in place of the standard FTP-75.  Please refer to TEST_FTP documentation for the reasons for this change. The results window then displays the individual cycle results along with a combined fuel economy as calculated in the following equation:

1/((.55/MPG_city)+(.45/MPG_Hwy))
